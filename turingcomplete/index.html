turingcomplete

As one of my hobbies, I collect stuff that is turing complete. The original idea comes from Andreas Zwinkau's “Accidentally Turing-Complete”. Here is his list:

  • C++ Templates
  • X86 MMU
  • Magic: The Gathering
  • HTML5 + CSS3
  • Minecraft
  • SQL (with Common Table Expressions and Windowing)
  • C Preprocessor (if executed in a loop)
  • Apache Rewrite Rules
  • Pokemon Yellow
  • Scala Type System
  • MediaWiki Templates
  • Little Big Planet
  • Server Side Includes
  • Sendmail
  • Border Gateway Protocol (BGP)
  • Excel

My additions:

  • printf (see also)
  • x86 MOV instruction (see MoVfuscator)
  • x86 XOR instruction (see MoVfuscator zu XORfuscator)
  • XSLT (see also)
  • sed (see also)
  • make (see also)
  • Haskell type system (see also)
  • make makro assignments (see also)
  • Java type system (see also)
  • Powerpoint (see also)
  • ELF meta data (see also)
  • MP4 (see also)
  • Dwarf Fortress (see also)
  • Star Craft (see also)
  • TypeScript type System (see also)
  • Age of Empires 2 Map Editor (see also)
